# Dreamina (即梦) API 接口文档分析

根据对 `curl.txt` 中请求记录的分析，主要的核心生成接口是 **Dreamina (即梦) AIGC 视频/图片生成接口**。

## 1. 任务创建接口 (AIGC 生成)

该接口用于提交视频或图片的生成任务。

- **URL**: `https://mweb-api-sg.capcut.com/mweb/v1/aigc_draft/generate`
- **Method**: `POST`
- **Content-Type**: `application/json`

### 请求参数 (Query Params)
| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `aid` | string | 应用 ID (示例: 513641) |
| `device_platform` | string | 设备平台 (示例: web) |
| `region` | string | 地区 (示例: SG) |
| `os` | string | 操作系统 (示例: windows) |

### 请求体 (Request Body) 核心字段
```json
{
  "submit_id": "c8ca7d21-0912-4e59-9ea2-e0074510dce6", // 任务唯一标识
  "extend": {
    "root_model": "dreamina_seedance_40", // 使用的模型版本
    "workspace_id": 0
  },
  "draft_content": {
    "version": "3.3.12",
    "component_list": [
      {
        "type": "video_base_component",
        "generate_type": "gen_video",
        "meta_list": [
          {
            "meta_type": "text",
            "text": "图片上增加一个小猫" // 提示词 (Prompt)
          },
          {
            "meta_type": "image",
            "material_ref": {
              "uri": "tos-alisg-i-wopfjsm1ax-sg/..." // 参考图 URI
            }
          }
        ]
      }
    ],
    "video_aspect_ratio": "16:9", // 视频比例
    "seed": 3116626725 // 随机种子
  }
}
```

## 2. 状态查询/历史获取接口

由于生成是异步的，需要通过此接口获取任务进度或结果。

- **URL**: `https://mweb-api-sg.capcut.com/mweb/v1/get_history_by_ids`
- **Method**: `POST`

### 请求体 (Request Body)
```json
{
  "history_ids": ["7147d37e-8f6b-795a-a4f8-efabd12614ab"], // 任务 ID 列表
  "http_common_info": { "aid": 513641 }
}
```

### 预期返回 (Response Structure)
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "history_list": [
      {
        "history_id": "7147...",
        "status": 2, // 状态码：1-进行中, 2-成功, 3-失败
        "result_url": "https://...", // 生成后的资源地址
        "cover_url": "https://..."
      }
    ]
  }
}
```

---

## 3. 关键 Header 说明
- `cookie`: 包含关键登录态 (`sessionid`, `sid_tt`)。
- `sign`: 请求签名 (JS 动态生成)。
- `did`: 设备指纹 ID。
- `loc`: 区域设置 (如 `SG`)。



## 4. 视频下载流程接口 (curl2.txt 分析)

在生成完成后，通过以下流程获取视频下载地址。

### 4.1 获取项目/素材详细信息
- **URL**: `https://mweb-api-sg.capcut.com/mweb/v1/get_local_item_list`
- **Method**: `POST`
- **说明**: 该接口用于根据素材 ID 获取包含下载链接在内的元数据。

#### 请求体 (Request Body)
```json
{
  "item_id_list": ["7623627365305961745"], // 需要下载的素材/视频 ID
  "pack_item_opt": {
    "scene": 1,
    "need_data_integrity": true
  },
  "is_for_video_download": true
}
```

### 4.2 视频 CDN 下载地址 (示例)
- **URL**: `https://v16-cc.capcut.com/b316656fbf3339cd32483d2f21b2a639/69d5cf46/video/tos/...`
- **Method**: `GET`
- **说明**: 这是最终的视频文件地址，通常由 `get_local_item_list` 接口返回。

#### 关键参数 (Query Params)
| 参数名 | 说明 |
| :--- | :--- |
| `mime_type` | 文件类型 (如 video_mp4) |
| `a` | App ID (513641) |

### 4.3 常用鉴权与校验 (各接口通用)
- **msToken**: 用于请求的安全校验，通常存在于 URL 参数或 Cookie 中。
- **X-Bogus / X-Gnarly**: 字节跳动系产品特有的反爬/安全签名参数，动态生成。



 关键接口详情：

  - 接口名称: AIGC 合规确认接口
  - URL: https://mweb-api-sg.capcut.com/mweb/v1/update_settings
  - 核心 Payload:
  {
    "custom_settings": {
      "aigc_compliance_confirmed": true
    }
  }
  - 作用: 该请求用于告诉服务器用户已经确认并同意了 AIGC 合规条款（通常针对人脸生成等敏感操作），提交后后续生成将不再弹出相关警告。
